# Project Summary

Dark technical English-language portfolio and blog platform for Blazej Pajor.

Stack:

- Next.js + TypeScript frontend/admin.
- Go REST API.
- Go MCP server with official MCP Go SDK.
- PostgreSQL.
- Docker Compose.
- GCP Compute Engine e2-micro.
- Cloudflare DNS/proxy.

Core constraints:

- hosting target below 50 PLN/month,
- strong security from frontend to infrastructure,
- admin panel for publishing,
- anonymous comments with moderation,
- MCP must be authorized and tool-scoped,
- SEO/GEO required for all public content.
